# Security Policy

## Supported Versions

Green Pulse is actively maintained. Security updates are provided for the `main` branch corresponding to the latest release.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We request that you do not publicly disclose any security vulnerabilities until we have issued a fix. 

To report a vulnerability, please contact the repository owner privately via email or direct message. 

We will respond to all security vulnerability reports within 48 hours. If the vulnerability is accepted, we will provide a timeline for a patch. We appreciate your efforts to responsibly disclose your findings, and we will make every effort to acknowledge your contributions.
